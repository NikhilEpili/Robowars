# Robowars Dashboard

A fully automatic scoring system for Robowars held in SAKEC, Mumbai.

## Overview
Robowars Dashboard is a lightweight, real‑time display and admin control panel for managing rounds, match lineups, and scoring during the event. Built for clarity in the arena and speed in the control room.

## Features
- Live match lineup display
- Automatic round progression
- Admin controls for advancing matches
- Dynamic ticker for ongoing matches
- Clean, arena‑friendly UI

## Tech Stack
- React + Vite
- Tailwind CSS

## Getting Started

### Install
1. Install dependencies:
   - npm install

### Run (Development)
- npm run dev

### Build
- npm run build

### Preview
- npm run preview

## Project Structure
- src/components — UI building blocks
- src/pages — App pages (Admin, Display, Match, Home)
- src/store.jsx — Tournament state and logic

## Usage Notes
- The admin view controls match advancement and round flow.
- The display view is intended for audience screens.
